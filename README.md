# Jaano

Jaano is a polished consumer-health product prototype for understanding packaged
foods sold in India. It combines a searchable food catalog, ingredient concerns,
side-by-side alternatives, and a persistent daily food diary.

## Run locally

```bash
npm install
npm run dev
```

Build the production bundle with:

```bash
npm run build
```

## Product scope

The included catalog is representative demo data, not a complete or
authoritative database. Product formulations and prices change, so a production
version should use current pack-label data, source citations, and a review
workflow. Health guidance is educational and non-diagnostic.
